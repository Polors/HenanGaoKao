# 志愿方案输出格式模板

按照以下格式输出最终方案。每个章节都不可省略。

## 【一、考生画像】

简要总结考生分数、位次、选科、兴趣、限制条件、风险偏好和家庭约束。

格式示例：
```
【考生画像】
- 总分：XXX 分
- 全省位次：第 XXXX 名
- 选科组合：XXX + XXX
- 外语语种：XXX
- 体检限制：XXX（无/有具体限制）
- 意向批次：XXX
- 意向专业方向：XXX
- 不可接受项：XXX
- 家庭预算：XXX
- 升学目标：XXX
- 风险偏好：XXX
- 地域就业打算：XXX
```

## 【二、政策与数据依据】

列出本次分析使用的官方政策、招生计划、招生章程、近年录取数据来源。无法确认的信息标注"需以 2026 年官方发布为准"。

格式示例：
```
【政策与数据依据】
- 招生政策：河南省教育考试院 2026 年招生政策文件（待核验/已确认）
- 批次设置：XXX（来源：XXX）
- 志愿填报单位：XXX（来源：XXX）
- 投档规则：XXX（来源：XXX）
- 选科要求：XXX（来源：XXX）
- 招生计划：XXX（来源：XXX）
- 录取数据：20XX-20XX 年河南录取数据（来源：XXX）
- 不确定项：XXX（需以 2026 年官方发布为准）
```

## 【三、核心判断】

说明该考生在河南 2026 年志愿填报中的大致定位，包括可冲层次、较稳层次、保底层次，以及应重点规避的风险。

## 【四、候选学校与专业池】

用表格列出候选志愿。表格字段：

| 学校名称 | 批次/志愿单位 | 专业或专业组 | 城市 | 办学性质 | 近年河南最低位次 | 2026计划变化 | 选科要求 | 风险等级 | 推荐理由 |
|---------|-------------|------------|------|---------|----------------|------------|---------|---------|---------|

说明：
- "近年河南最低位次"填近 3 年数据，标注年份
- "2026计划变化"填增加/减少/新增/待核验
- "风险等级"填冲/稳/保
- 所有数据标注来源，无法核验的标注"待核验"

## 【五、方案 A：稳妥保守型】

按志愿顺序输出，说明每个志愿为什么放在这个位置。

每个志愿格式：
```
志愿 X：学校名称 - 专业/专业组
- 城市：XXX
- 办学性质：XXX
- 近年位次：XXX（年份范围）
- 风险等级：保/稳
- 推荐理由：XXX
- 专业调剂风险：XXX
- 限制提示：高收费/中外合作/校区差异/体检限制/单科限制（如有）
- 放在此位置的原因：XXX
```

方案末尾说明冲稳保比例。

## 【六、方案 B：均衡型】

同方案 A 格式。冲稳保比例应体现均衡特点。

## 【七、方案 C：冲刺型】

同方案 A 格式。冲稳保比例应体现冲刺特点，但必须有合理保底。

## 【八、退档、滑档与调剂风险检查】

逐项检查每套方案：

1. 是否满足选科要求
2. 是否有体检限制
3. 是否有单科成绩限制
4. 是否有外语语种限制
5. 是否存在高收费专业
6. 是否存在不同校区差异
7. 是否存在专业组内调剂到不可接受专业的风险
8. 是否有滑档风险

每项检查结果格式：
```
检查项 X：XXX
- 结果：通过/存在风险
- 详情：XXX
- 建议：XXX
```

## 【九、最终建议】

给出最推荐的一套方案，并说明理由。提醒用户最终填报前必须再次核对：

1. 河南省教育考试院公布的 2026 年招生计划
2. 志愿填报规则
3. 目标高校招生章程
4. 志愿填报系统中的实时信息

格式示例：
```
【最终建议】
最推荐方案：方案 X（XXX型）
推荐理由：XXX

重要提醒：
- 填报前请务必核对河南省教育考试院公布的 2026 年最新招生计划
- 请核对目标高校 2026 年招生章程中的选科要求、体检要求、单科要求
- 请在志愿填报系统中确认实时信息
- 本方案基于历史数据和当前已知信息，实际录取结果受多种因素影响
```
